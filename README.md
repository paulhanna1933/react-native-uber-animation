# React Native Uber UI with Animation
A React Native Uber UI clone app with animation using create-react-native-app

# Tools
* create-react-native-app cli
* native-base
* react-navigation


# Start App
Clone repo, install, cd into folder and run:
```git
npm install
npm run ios
```
